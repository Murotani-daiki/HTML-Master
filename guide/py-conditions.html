条件分岐

プログラムに「判断」をさせ、処理の流れを切り替えよう

条件分岐とは？

「もし〜ならこうする、そうでなければこうする」というロジックを担うのが条件分岐です。ゲームの判定やユーザーの入力チェックなど、あらゆる場面で使用されます。

基本の if 構文

条件式の後に `:`（コロン）を書き、次の行からは字下げ（インデント）をします。

Python 
score = 85
if score >= 80:
    print("合格です！")

⚠️ インデントが命！

Pythonでは「どこからどこまでが if 文の範囲か」を、波カッコ `{ }` ではなく、**字下げの量**で判断します。通常、半角スペース4つ分を使います。これがズレるとエラーになるので注意しましょう。

else と elif

「さもなくば」を表す `else` や、さらに条件を重ねる `elif`（else if の略）を組み合わせることができます。

Python 
score = 75
if score >= 80:
    print("素晴らしい！")
elif score >= 60:
    print("合格！")
else:
    print("不合格...")

比較演算子

  • `==` 等しい / `!=` 等しくない
  • `>` より大きい / `<` より小さい
  • `>=` 以上 / `<=` 以下

まとめ

条件分岐はプログラムに「知性」を与える第一歩です。インデントを正しく使いこなして、複雑なロジックを美しく組み立てましょう。
