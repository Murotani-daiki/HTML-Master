データ型

JavaScriptが扱う情報の「種類」をマスターしよう

データには「型」がある

プログラミングで扱うデータには、それぞれ決まった種類（型）があります。JavaScriptは、数値として計算するのか、文字として繋げるのかを、このデータ型を見て判断します。

String (文字列)

名前や文章など、引用符 `"` や `'` で囲まれたテキストデータです。

Number (数値)

整数や小数。計算に使用される数字データです。

Boolean (真偽値)

`true`（真）か `false`（偽）の2つの値しか持たない型です。

1. 文字列 (String)

文字列は自由に文字を繋げることができます。また、`backtick` ( ` ) を使うと、文字列の中に変数を埋め込むこともできます。

const name = "JavaScript";
console.log(`Hello, ${name}!`); // テンプレートリテラル

2. 配列 (Array)

複数のデータをひとまとめにして管理する「棚」のようなデータ型です。各データには `0` から始まる番号（インデックス）でアクセスします。

const colors = ["red", "blue", "yellow"];
console.log(colors[0]); // "red" を表示

3. オブジェクト (Object)

データに名前（キー）をつけて管理する型です。関連する情報をまとめて扱うのに非常に便利です。

const user = {
    name: "Taro",
    level: 10
};
console.log(user.name); // "Taro" を表示

まとめ

データ型を正しく理解することは、意図しない計算ミス（例：1 + "1" が "11" になる）を防ぐために不可欠です。今扱っているのがどの型なのかを意識しながらコードを書いてみましょう。型の確認には `typeof` 演算子が役立ちます。
