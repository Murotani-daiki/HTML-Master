JavaScriptとは何か？

JavaScriptは、ウェブサイトにインタラクティブな動きを加えるためのプログラミング言語です。1995年に誕生して以来、ウェブの進化とともに成長し、現在ではブラウザ上だけでなく、サーバー（Node.js）やアプリ開発など、あらゆる場所で使用されています。

HTMLが「骨格」、CSSが「見た目」なら、JavaScriptは「筋肉」や「神経」に例えられます。ボタンをクリックした時の動作、データの読み込み、アニメーションの制御など、ユーザーの操作に反応してウェブページを変化させるのがJavaScriptの役割です。

基本文法：変数と定数

プログラミングの基本は、データを一時的に保存する「変数」を扱うことです。モダンなJavaScriptでは、値を書き換えられる変数には `let`、一度決めたら変えない定数には `const` を使用します。

const name = "JavaScript Master";
let score = 100;

score = score + 10; // 値の更新
console.log(name + "のスコアは" + score);

データ型と演算

JavaScriptが扱うデータにはいくつかの「型」があります：

  • 文字列 (String): "あいうえお" のようなテキスト。
  • 数値 (Number): 10 や 3.14 などの数字。
  • 真偽値 (Boolean): true（正しい）か false（間違い）かの2択。
  • 配列 (Array): 複数のデータをまとめて管理する棚のようなもの。

DOM操作：ページを書き換える

JavaScriptの真髄は「DOM（Document Object Model）操作」にあります。これは、JavaScriptを使ってHTML要素を自由自在に書き換える仕組みのことです。例えば、ボタンを押した時に文字の色を変えたり、新しいメッセージを表示したりするのは、すべてこのDOM操作によって行われています。

まとめ

JavaScriptは非常に奥が深い言語ですが、まずは「コードを書いて、画面が変わる」という楽しさを知ることが大切です。プログラミングの基本ロジックは他の言語とも共通しているため、JavaScriptをマスターすることは、エンジニアとしての確かな一歩になります。
