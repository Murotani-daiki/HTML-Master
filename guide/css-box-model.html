ボックスモデル

CSS を理解する最大の鍵：全ての要素は「四角い箱」でできている

ボックスの4層構造

ウェブページの全ての要素は、4つの境界線を持つ長方形として扱われます。これを正確に操れるようになると、思い通りのレイアウトが作れるようになります。

Margin (外側の余白)
Border (枠線)
Padding (内側の余白)
Content (中身)

1. Content (コンテンツ)

テキストや画像が実際に表示される部分です。`width` や `height` で大きさを指定します。

2. Padding (パディング)

コンテンツと枠線の間の**内側の余白**です。文字が枠にくっつくのを防ぎ、要素の中にゆとりを持たせます。

3. Border (ボーダー)

要素を囲む**枠線**です。太さ、線種（実線や点線）、色を指定できます。

4. Margin (マージン)

枠線の外側にある、他の要素との**間隔**です。要素同士がくっつきすぎるのを防ぎ、ページ全体のバランスを整えます。

注意点：全体の幅はどう決まる？

デフォルトでは、要素の合計幅は `width + padding + border` になります。マージンは要素の外側に計算されます。

`box-sizing: border-box;` という設定を使うと、padding も border も含めて `width` ぴったりに収まるようになり、レイアウトの計算が非常に楽になります。現代のWeb制作では必須の設定です。

まとめ

「内側の余白は Padding、外側の間隔は Margin」と覚えましょう。この4つのプロパティをマスターすることが、プロのような洗練されたデザインへの近道です。
