効率化

ループ処理

コンピュータの得意技：めんどくさい繰り返しをプログラムで解決

繰り返しを自動化する

「100個のデータを表示する」「同じ計算を10回繰り返す」といった作業を人間が手で行うのは大変です。プログラミングの「ループ」を使えば、たった数行のコードで何千回もの繰り返しを瞬時に終わらせることができます。

for 文の基本

最も一般的な繰り返し構文です。カッコの中に「始まり、終わり、増え方」を記述します。

for (let i = 1; i <= 5; i++) {
    console.log(i + "回目の繰り返しです");
}

このコードは、i が 1 から始まり、5 になるまで、1ずつ増えながら実行されます。

配列とループ

ループは配列（データのひとまとめ）を処理する時に最も力を発揮します。

const users = ["Alice", "Bob", "Charlie"];
for (let i = 0; i < users.length; i++) {
    console.log("こんにちは、" + users[i] + "さん！");
}

無限ループに注意！

終わりの条件を書き忘れると、プログラムが永遠に繰り返され、ブラウザがフリーズしてしまいます。必ず「いつ終わるか」を指定するようにしましょう。

まとめ

ループは、プログラミングによる「効率化」の核心です。配列とループを組み合わせれば、大量の情報を一瞬で処理する高度なサービスを作れるようになります。
