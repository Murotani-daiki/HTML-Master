イベント

ボタンが押されたら？：ユーザーの「きっかけ」を機能に変える

Webを「動かす」きっかけ

「ボタンをクリックする」「テキストボックスに入力する」「ページを最後までスクロールする」など、ブラウザ上で発生するあらゆる動作のことを「イベント」と呼びます。

JavaScriptはこれらのイベントを待ち構え、発生した瞬間に特定のプログラムを実行することで、Webサイトにインタラクティブな動きを与えます。

主要なイベント一覧

click

要素がクリックされた時

input

文字が入力された時

submit

フォームが送信された時

addEventListener の魔法

要素に対して「イベントの聞き耳を立てる」命令を出します。第一引数にイベント名、第二引数に実行したい処理を書きます。

const btn = document.querySelector("#my-btn");

btn.addEventListener("click", () => {
    alert("ボタンがクリックされました！");
});

イベントオブジェクト

「どこの座標でクリックされたのか」「どのキーが押されたのか」など、イベントに関する詳細な情報を受け取ることもできます。

document.addEventListener("keydown", (event) => {
    console.log("押されたキーは:", event.key);
});

まとめ

イベントを使いこなせば、ただの表示物だったWebページが、ユーザーと対話する「アプリケーション」へと進化します。どんなアクションがどんな魔法（機能）を引き起こすのか、ワクワクしながら実装してみましょう！
