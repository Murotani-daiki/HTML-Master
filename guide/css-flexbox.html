Flexbox レイアウト

直感的に、美しく要素を並べるためのレイアウト・ツールキット

Flexbox とは？

要素を一行の中に横並びにしたり、位置を細かく調整したりするための、現代Web制作で最も使われているレイアウト手法です。わずか数行の指定で、中央揃えや均等配置が魔法のように実現できます。

1. 横並びにする

並べたい要素の「親（コンテナ）」に対して `display: flex;` を指定するだけで、要素が横に並びます。

.container {
    display: flex;
}

2. 配置のコントロール

以下の強力なプロパティを使って、位置を自由に変えられます。

justify-content（横向きの配置）

  • `center`: 中央に寄せる
  • `space-between`: 両端に寄せて、間を均等に空ける

align-items（縦向きの配置）

  • `center`: 上下の中央に揃える（超便利！）
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100px;
}

3. フレキシブルな幅調整

`gap` プロパティを使うことで、要素同士の間隔だけを簡単に指定できます。また、`flex: 1;` と書くことで、余ったスペースを要素で分け合うことも可能です。

まとめ

Flexbox を覚えれば、もうレイアウトで悩むことはありません。まずは `display: flex`, `justify-content`, `align-items` の3点セットを使いこなせるようになりましょう。Webサイトの構造作りが格段に楽しくなります。
