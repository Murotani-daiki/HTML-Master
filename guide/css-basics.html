CSSの基礎知識

ウェブサイトに美しさと使いやすさを加えるデザインの技術

CSSとは何か？

CSS（Cascading Style Sheets）は、HTMLで作られたウェブページの「見た目」を指定するための言語です。HTMLが文書の構造（見出しや段落など）を担当するのに対し、CSSはその文字の色、大きさ、配置、背景デザイン、アニメーションなどを担当します。

「Cascading（カスケード）」には「階段状に流れる」という意味があり、CSSでは複数のスタイル指定が重なった際に、どのルールを優先的に適用するかという仕組みが備わっています。これによって、サイト全体のスタイルを一括で管理でき、効率的なデザイン制作が可能になります。

CSSの基本構造：セレクタとプロパティ

CSSは基本的に「どの要素に」「何を」「どう変えるか」という形式で記述します。この「どの要素に」をセレクタ、「何を」をプロパティ、「どう変えるか」を値と呼びます。 

h1 {
    color: blue;
    font-size: 24px;
}

上記の場合、`h1`（セレクタ）に対して、`color`（プロパティ）を `blue`（値）にする、という命令になります。これらを波括弧 `{ }` で囲むのがCSSの基本ルールです。

ボックスモデルの理解

CSSを理解する上で最も重要な概念のひとつに「ボックスモデル」があります。ウェブページ上の全ての要素は、目に見えない「四角い箱（ボックス）」として扱われます。このボックスは、以下の4つの層で構成されています：

  • Content (コンテンツ): 文字や画像自体が表示される領域。
  • Padding (パディング): コンテンツと枠線の間の内側の余白。
  • Border (ボーダー): パディングの外側を囲む枠線。
  • Margin (マージン): 枠線の外側にある、他の要素との間隔。

このボックスモデルの仕組みを理解することで、要素同士を正確に配置し、バランスの取れたレイアウトを実現できるようになります。

色の指定方法

CSSでは、色は様々な方法で指定できます。代表的なものに以下の3つがあります：

  • 色名: `red`, `blue`, `white` などの名前で直接指定。
  • HEX値: `#ffffff` や `#38bdf8` のような16進数で指定。
  • RGB/RGBA: `rgb(255, 0, 0)` や、透明度を含めた `rgba(0, 0, 0, 0.5)` で指定。

まとめ

CSSをマスターすることで、シンプルなHTML文書を、洗練された「ウェブサイト」へと昇華させることができます。最初は覚えることが多いように感じるかもしれませんが、基本のセレクタとプロパティさえ押さえれば、あとはそれらの組み合わせです。
