セレクタと詳細度

「どの要素」にスタイルを当てるか、その優先順位のルールを学ぼう

セレクタの種類

CSSには、HTML要素を選択するための様々な方法があります。代表的なものをマスターしましょう。

1. 要素セレクタ

HTMLタグの名前で直接指定します。ページ内の全てのそのタグに適用されます。

CSS 
p { color: gray; }

2. クラスセレクタ

特定の名前がついたグループに適用します。最も頻繁に使われるセレクタです。ドット `.` で始めます。

CSS 
.main-text { font-size: 18px; }

3. IDセレクタ

ページ内でたった一つの要素だけに適用します。ハッシュ `#` で始めます。

CSS 
#header { background: #333; }

詳細度（優先順位）

同じ要素に対して複数のスタイルが競合したとき、どちらが優先されるかを決定するのが「詳細度」というスコアです。

セレクタ 詳細度の強さ 例
インライン 最強 style="..."
IDセレクタ 強 #menu
クラスセレクタ 中 .button
要素セレクタ 弱 div, p, h1

例えば、同じ段落に対して要素セレクタで `blue` 、クラスセレクタで `red` が指定されていた場合、クラスセレクタの方が優先度が高いため、文字は `red` になります。

後勝ちの法則

詳細度が全く同じだった場合は、CSSファイルの下の方（後から書かれた方）が優先されます。

まとめ

セレクタを使い分けることで、ページ全体の共通デザインと、特定の場所だけの特別デザインを効率よく作り分けることができます。思った通りに色が反映されないときは、この「詳細度」を確認してみましょう。
