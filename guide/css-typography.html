タイポグラフィと単位

心地よい「読み体験」を作るためのフォント設定と、柔軟な単位の使い分け

文字をデザインする

Webサイトの情報の9割はテキストです。タイポグラフィ（文字の配置やデザイン）を整えることは、サイトの質を左右します。

p {
    font-size: 16px;
    font-weight: 600;   /* 太さ */
    line-height: 1.8;   /* 行間 */
    letter-spacing: 0.05em; /* 字間 */
    color: #333;
}

Web制作で使う「単位」の使い分け

CSSには多くの単位がありますが、特によく使われるものを覚えましょう。

px (ピクセル)

絶対的なサイズを指定します。「16px」と書けばどんな環境でも16ピクセルですが、画面の大きさに合わせた調整には向きません。

rem (レム)

ルート（HTML要素）のフォントサイズを基準にする**相対的な単位**です。レスポンシブ対応しやすいため、現代の主流です。

% (パーセント)

親要素に対する割合を指定します。ボタンの幅を「100%」にして横いっぱいに広げたい時などに重宝します。

vw / vh

画面（ビューポート）の幅や高さに対する割合です。ブラウザの全幅いっぱいに要素を広げたい時に使います。

行間（line-height）の魔法

文字がぎゅうぎゅうに詰まっていると、ユーザーは疲れてしまいます。行間を `1.5` 〜 `2.0` 程度に少し広げるだけで、一気に「読みやすい本格的なサイト」になります。

まとめ

「とりあえず px」ではなく、文字サイズには `rem` 、余白調整には `px` や `rem` 、レイアウトには `%` や `vw` といった使い分けを意識しましょう。柔軟な単位使いが、レスポンシブなWebサイトを作る鍵となります。
