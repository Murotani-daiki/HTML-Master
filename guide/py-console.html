出力とデバッグ

print()：プログラムの実行結果を画面に表示しよう

print関数とは？

プログラムの中で起きていることを私たちの目に見える形にするのが `print()` 関数です。変数の値が正しく計算されているか、プログラムがどこまで実行されたかを確認するための、プログラマーにとって最も身近な武器です。

基本の使い方

丸カッコの中に表示したいデータ（文字列や数値、変数）を入れます。

Python 
print("Hello Python")
print(123)
x = "Master"
print(x)

f-string（便利な埋め込み）

Python 3.6から導入された「f文字列」を使うと、文章の中に変数をとても簡単に埋め込むことができます。文字列の前に `f` をつけ、入れたい変数を `{ }` で囲みます。

Python 
name = "Taro"
score = 80
print(f"{name}さんのスコアは{score}点です")

カンマで複数の値を表示

カンマ `,` で区切ることで、複数の情報を同時に出力できます。自動的にスペースが挿入されるので便利です。

Python 
print("結果は:", 100)

まとめ

「バグかな？」と思ったら、迷わず `print()` を使って中のデータを確認しましょう。プログラムが見える化されることで、理解がぐっと深まります。
