色と背景

サイトの印象を決める色彩設計と、豊かな背景表現をマスターしよう

CSSでの色の指定方法

CSSでは、主に以下の3つの方法で色を表現します。状況に合わせて使い分けましょう。

色名
skyblue
HEX値
#38bdf8
RGB
rgb(56, 189, 248)
RGBA
rgba(..., 0.5)

1. HEX値（16進数）

`#` から始まる形式で、Webデザインで最も一般的です。赤、緑、青の強さをそれぞれ16進数（0-F）で表します。

2. RGB / RGBA

`rgb(255, 0, 0)` のように0〜255の数値で指定します。`rgba` の `a` は「透明度」を表し、0.0（完全透明）から1.0（不透明）までの値で指定できます。重ね合わせのデザインに重宝します。

背景（background）の設定

単色だけでなく、画像やグラデーションを設定することで、サイトの質感が一気に向上します。

背景画像

body {
    background-image: url("bg.jpg");
    background-size: cover; /* 画面いっぱいに広げる */
    background-position: center;
}

グラデーション

最近の流行は `linear-gradient` を使った美しい変化です。

background: linear-gradient(135deg, #38bdf8, #818cf8);

まとめ

色には「視認性を高める」「ブランドイメージを伝える」といった重要な役割があります。適切な色の指定と背景設定をマスターして、ユーザーを魅了する美しいページを作りましょう。
