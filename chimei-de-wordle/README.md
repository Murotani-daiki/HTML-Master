# ⛩️ 日本の地名でWordle

日本の地名を使ったWordleゲームです。5文字のローマ字で書かれた日本の地名を6回以内に当てましょう！

## 🎮 主な機能

### ✅ 実装済みの機能
- **Wordleゲームロジック**: 5文字のローマ字地名を6回以内に推測
- **色分けヒント**: 🟩正しい位置 / 🟨含まれるが位置違い / ⬛含まれない
- **オンスクリーンキーボード**: タップ/クリックで入力可能（物理キーボードも対応）
- **キーボード色追跡**: 使用した文字の結果がキーボードに反映
- **Google Map表示**: 正解発表時にその地名の場所をGoogle Mapで表示
- **統計機能**: プレイ回数、勝率、連勝記録、推測分布を記録
- **シェア機能**: 結果を絵文字でクリップボードにコピー
- **アニメーション**: タイルフリップ、シェイク、バウンス、紙吹雪、桜の花びら
- **レスポンシブデザイン**: モバイル/タブレット/PC対応
- **156個の日本の地名**: 都道府県名、市区町村名、有名地区名を収録

### 🗾 収録地名の例
- 都道府県: TOKYO, OSAKA, KYOTO, AICHI, CHIBA...
- 観光地: NIKKO, ATAMI, OTARU, BEPPU, GINZA...
- 市区町村: CHOFU, EBINA, KOMAE, SABAE, TSURU...

## 🎯 遊び方

1. 5文字のローマ字（A-Z）で日本の地名を入力
2. Enterキーで推測を確定
3. タイルの色を手がかりに次の推測を行う
4. 6回以内に正解を見つける
5. 正解/失敗後、Google Mapでその場所が表示される

## 💾 データ保存

- **localStorage** を使用してブラウザに統計データを保存
  - `chimeidle-stats`: プレイ統計（回数、勝率、連勝、分布）
  - `chimeidle-visited`: 初回訪問フラグ
